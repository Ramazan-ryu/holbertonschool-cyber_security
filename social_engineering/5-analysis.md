# Social Engineering Project — Analysis

## 1. Which SET menu option creates a payload?

The payload is created using:
Social-Engineering Attacks → Create a Payload and Listener

This option allows the attacker to generate a malicious payload and automatically start a listener to receive incoming connections.

---

## 2. Which SET menu option sends emails?

Emails are sent using:
Social-Engineering Attacks → Mass Mailer Attack

This option is used to send phishing or simulation emails to target users as part of a social engineering campaign.

---

## 3. Why did we host the payload on a web server instead of attaching it directly?

We hosted the payload on a web server because email security systems often block executable (.exe) attachments. Hosting the file on a web server and sending a download link helps bypass basic email filters. It also makes the message appear more legitimate, since links are commonly used to share documents.
